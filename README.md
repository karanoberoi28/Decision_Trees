# Decision_Trees
This repository contains code to build/learn decision trees from scratch.

This repo serves as a learning tutorial for coding a Decision Tree from scratch in Python using just NumPy and Pandas. And here is the accompanying [blog post](https://www.sebastian-mantey.com/code-blog/coding-a-decision-tree-from-scratch-python-p1-introduction).
